# 🚀 Calculadore | Calculator

Aplicação desktop de uma calculadora simples desenvolvida em Python.

---

## 📌 🇧🇷 Descrição

Aplicação desktop desenvolvida em **Python 3** que realiza operações matemáticas básicas, como:

- ➕ Adição  
- ➖ Subtração  
- ✖️ Multiplicação  
- ➗ Divisão  

O projeto tem como objetivo reforçar conceitos fundamentais de programação, como:

- 🧠 Lógica de programação  
- 🔢 Manipulação de operadores matemáticos  
- 🏗️ Organização de código
